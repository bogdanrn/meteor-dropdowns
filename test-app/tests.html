Basics

{{#dropdownTrigger name="testDropdown1"}} {{/dropdownTrigger}} {{#dropdown name="testDropdown1"}}

{{/dropdown}}
{{#markdown}} Simply use the templates `dropdownTrigger` and `dropdown` in order to wrap the triggering element and the dropdown: ```html {{#dropdownTrigger name="dropdownTest"}} {{/dropdownTrigger}} {{#dropdown name="dropdownTest"}}

Hello world

{{/dropdown}} ``` {{/markdown}}
